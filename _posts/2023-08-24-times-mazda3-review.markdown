# 旅行のお供
![Mazda3ファストバックと瀬戸大橋](/images/mazda3_1.jpg)

8月上旬, 友人と3人で一泊二日のドライブ旅行に出かけました. 今回はガソリン代が無料, かつ比較的新しい車両を安く借りられるということでタイムズカーシェアを利用してみました. 

タイムズカーシェアでは三段階の車両クラス(ベーシック・ミドル・プレミアム)により車種・料金区分がありますが, 今回は最も利用料金の安いベーシッククラスの中で最も車格の高いMazda3 ファストバックを借りてみました. 車での移動時間が長くなることを考えると車体の大きさや快適性の高さが重要になると感じたため, その点で期待してのチョイスです. 

# クルマの詳細
前置きが長くなりましたが, 今回お借りしたMazda3 ファストバックのスペックです. 

| --- | --- |
| エンジン | 1.5L直列4気筒自然吸気 P5-VPS 111PS (6000rpm) /146Nm (14.9kgf-m) |
| トランスミッション | 6AT(マニュアルモード付) |
| 駆動方式 | 前輪駆動 (FF) |
| サスペンション 前/後 | ストラット/トーションビーム |
| タイヤサイズ | 205/60R16 |
| 全長/全幅/全高 (mm) | 4460/1795/1440 |
| 車両重量 | 1340kg |

タイムズカーで借りられる車両は, カタログモデルには存在しない15Cという法人・レンタカー向け廉価グレードで, 装備の一部が簡素化されています. たとえば, カタログモデルの場合ホイールは全車アルミホイールですが, このグレードの場合はスチールホイールにフルキャップが被せられています. しかし, このグレードでもマツダ特有のナビゲーションシステムであるマツダコネクトや, 追従式クルコン, ブラインドスポットモニター, プリクラッシュブレーキなどの先進安全装備, ヘッドアップディスプレイやLEDヘッドランプなどのドライビングを快適にする装備は一通り揃っており, カーシェアの車として乗る分には多くの人は不満を覚えることはないでしょう. 

ということで, ここからはMazda 3の気に行ったポイントと気になったポイントをいくつか取り上げてみたいと思います. 

![Mazda3 15Cのホイール](/images/mazda3_wheel.jpg)

# 「人馬一体」は伊達じゃないかも
このクルマで一番素晴らしいと感じた点は, 人間の操作に対して車が非常に素直に動く点です. アクセルペダルも開度に応じて非常にリニアにスロットルが反応していく感覚がありますし, ブレーキは踏めば踏んだだけ効き, 制動力が微調整しやすく非常に車両の挙動をコントロールしやすいです. やや径が大きく, 手応えが重めなステアリングも, 反応が滑らかかつ素早く狙ったラインをしっかりとトレースしてくれます. ロール感も抑えられていてアールが大きいカーブでも全く怖くないです. 体の動きとドライバーの思考にしっかりと車が追従してくれるのを感じ取れます. そのおかげか長時間の運転でもあまり疲れを感じませんでした. 

マツダは「人馬一体」をスローガンに長年車づくりを続けていますが, その努力がしっかりと感じ取れる, 人間の意思や動きに忠実なフィーリングが素敵でした. 

# (たぶん)だれが見てもカッコいいデザイン
伸びやかでノーズが長いデザインは美しいです. 日本メーカーのCセグメントの中で個人的にはトップレベルに美しいと感じます. 
ただしクォーターウィンドウが存在せず, 極太のCピラーがそびえたっている点から想像できる通り, 斜め後方の視界は非常に悪いです. しっかりとデザインの代償を受けています. 前方はAピラー付近の死角も少ないので, 前方の確認はけっこう楽ですが巻き込み確認や車線変更時, 駐車時などは気になる点ではあります. 
反面, 車両感覚は意外とつかみやすいです. 前述した通り前方視界が優れているのと, リアフェンダーが若干張り出していて後輪の位置がサイドミラー越しに見やすいのもあって, ノーズの長さにさえ気をつければぶつける心配はせずに済みます. 旅行中何度か狭い場所ですれ違いをする機会がありましたが, 全幅1795mmと決して小さくはないボディでもギリギリまでタイヤを溝に寄せつつ対向車に道を譲るのにもそれほど苦労しませんでした.  

![Mazda3 iPhoneにて3倍望遠で撮影](/images/mazda3_side.jpg)

# 1.5Lはアンダーパワーだけどブン回せば！？
今回お借りした車両に搭載されている1.5Lエンジンは最高出力111PSとお世辞にも高性能といえるエンジンではありません. 正直重量1340kgとCセグメントハッチバックとして立派な車体を動かすには絶対的なパワーでは非力です. 大人3人と軽めの荷物が乗った状態で, 緩く長い上り坂などを登る際はアクセル開度が低いと, どんどん失速していきます. 

ところが, しっかりとアクセルを踏み込むと素早くキックダウンし, とくに中回転域で見られる豊かなトルクと共にヌルヌル加速していきます. ヌルヌルという表現から感じられる通り速くはないですが滑らかなフィーリングです… 乾いた甲高いサウンドは不快感やガサツさがほぼありません.  

アクセル開度を抑えると正直かったるいですが, しっかりとアクセルを積極的に踏むドライバーであれば, 余裕のなさは感じるにしろむしろ限られたパワーを使い切っている感覚が心地よいとさえ感じるのではないでしょうか. 踏めば極端に遅いとは思わなかったですし, 平地であれば高速での追い越しも普通にこなせます.  
MTモードで走行している際はなかなかシフトチェンジはサボれませんが, 車の特性を読み取って自らの手で車のトルクバンドをしっかり保って走行する感覚が面白かったです. 

タイムズカーで借りられるのはAT車のみですが, かつてカタロググレードに存在していた同じエンジンの3ペダルMT仕様もいつか乗ってみたくなりました…

# 高い静粛性
車内の静粛性の高さも魅力です. 専用の測定器具などは持っていないため具体的な数値は計測していませんが, 車内での会話で声を張り上げる必要は当然ありませんし, オーディオを小さめの音量でもしっかり楽しむことができます. ただ, エンジンをしっかり回せばそれ相応のノイズはしっかり聞こえます. Mazda3のようなややスポーティーなクルマが好きな層にはウケそうですが, 同乗していた特にクルマ好きではない友人はやや面食らっていたのでここは好みが分かれてくるポイントです. 

# 後部座席は割り切りが必要
後部座席のレッグルームがそこまで広くないのと, 道路の継ぎ目を超えた際などのリアサスペンションの突き上げ感がフロントに比べやや強めなのが気になります. クォーターウィンドウがないことで閉塞感もやや大きいので, 車酔いする人は前席に座るのがよいでしょう. (運転席だけでなく助手席にもシートリフターがついているので助手席に座る人も好みのポジションをとりやすいです) シートバックの角度など, シート自体の出来は結構いいだけに少し残念にも感じます. これは居住性とはあまり関係ないかもしれないですが, ドアの開閉音もリアドアは遮音材等が少ないのかフロントドアに比べペラペラした音がします. フロントドアが良すぎるのかもしれませんし, 大体の車がそうといえばそうなのですが… 

3~4名乗車も想定されるCセグメントとして標準的なサイズのハッチバックではありますが, 多人数ドライブを楽しむクルマというよりは1~2名乗車に重点を置いたドライバーズカーとしての側面がより大きい印象を受けます. 

# キーレスエントリーのボタンが使いにくい
細かい点ではありますが, キーレスエントリーのボタンが非常に使いにくかったです. キー自体は最近流行りのキーを携帯した状態でドアハンドルのタッチセンサーに触れることで解錠/施錠できるタイプのスマートキーと近い形状なのですが, タイムズカーで借りられるMazda3はスマートキーではなくキーのボタンを押して操作するキーレスエントリーです. これは全く問題ないのですが, 他社のキーレスエントリーと違いどちらが解錠か施錠かわかるようなポッチや形状の違いがないためボタン位置を覚えるまではキーを目視しないと一発で目当ての操作ができませんでした. この仕様であれば, 一つのボタンが解錠/施錠を兼任するトグル式のボタンでも良いような気がします. 

# マツダコネクトはタッチ操作に対応してほしいなぁ…
ダッシュボード中央上部からぬるりと出ているナビゲーションシステムのマツダコネクトのモニターですが, タッチ操作ができそうな顔をして実はタッチパネルではありません. ではどのように操作するのかというとセンターコンソールにあるコマンダースイッチを動かして操作するのですが, これが非常に面倒です. カーソル移動やスクロールは比較的すぐ慣れますが, 文字入力までコマンダー操作を強いられるのは全く直観的ではありません. マツダ以外にもさまざまなメーカーで採用されてきたコマンダースイッチも面白いのですが, スマートフォンの操作に慣れた人が多い現在ではタッチパネルでも操作できるほうがさまざまなユーザーに受け入れられるでしょう. 

なお, Apple CarPlayやAndroid Autoといったスマホ用のドライビングサポート機能はしっかりと対応しているため, どうしても慣れない！という方はこっちでカーナビとしての機能を完結させてしまうのもアリです. 私はほぼ全ルートでAndroid Autoを使用していました…
![マツダコネクト Android Auto 使用例](/images/mazda3_androidauto.jpg)

# 総評
今回のMazda3 ファストバックはスタイリッシュなデザインと1.5Lエンジンのアンダーパワーながらも楽しい特性,直観的でコントロールしやすいドライブフィールと静粛性の高い車内が魅力の一台でした. 反面, 後部座席の居住性がすごくいいわけではなく, あくまでフロントシート優先の設計がなされているという点や独特の操作感を持つマツダコネクトなどは乗る人を選ぶポイントだと感じます. とはいえ, 運転していて疲れにくいという点やドライブフィールの良さから, 長距離走行に非常に向いている車だと思います. ドライブ好きの人にぜひ乗って欲しい一台でした.
